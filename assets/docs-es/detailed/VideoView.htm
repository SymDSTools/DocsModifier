Vista de Video

Crear una Vista de Video usando la función CreateVideoView del objeto de la app:

 player = app.CreateVideoView( width, height, options );

Estos controles le permiten a su programa reproducir archivos de vídeo almacenados a nivel local en el teléfono/tableta, y también de vídeos desde fuentes remotas (Streaming).

Usa SetFile para establecer el archivo de video o la dirección del video streaming.

 player.SetFile( "http://www.nasa.gov/multimedia/nasatv/NTV-Public-IPS.m3u8" );

Cuando el video esté listo para reproducir, la función callback OnReady será llamada, luego puedes usar Play para empezar a reproducir el video.

Si el video no puede ser reproducido, la función callback OnError será llamada. Usa SetOnReady y SetOnError para establecer los nombres de las funciones callback OnReady y OnError.

El siguiente ejemplo muestra como streaming un video.

Ejemplo - Video Streaming

function OnStart()
{
  lay = app.CreateLayout( "Linear", "FillXY,VCenter" );

  app.ShowProgress( "Cargando..." );

  player = app.CreateVideoView( 0.9, 0.4 );
  player.SetOnReady( player_OnReady );
  player.SetOnError( player_OnError );
  lay.AddChild( player );

  player.SetFile( "rtsp://wowzaec2demo.streamlock.net/vod/mp4:BigBuckBunny_115k.mov" );
  app.AddLayout( lay );
}

function player_OnReady()
{
  app.HideProgress();
  app.ShowPopup( "OnReady" );
  player.Play();
}

function player_OnError()
{
  app.HideProgress();
  app.ShowPopup( "OnError" );
}

 GetHeight()

 Play()
 Pause()
 Stop()
 IsReady()
 IsPlaying()

 GetPosition()
 GetDuration()
