Reproductor de Música

El objeto MediaPlayer (Reproductor de Música) puede ser usado para reproducir un archivo de sonido del celular o tableta.

Crea un objeto MediaPlayer usando el método CreateMediaPlayer de los objetos de la app:

 player = app.CreateMediaPlayer();

Usa el método SetFile (establecer archivo) del objeto de MediaPlayer para establecer el archivo de sonido a reproducir. Los tipos de sonidos soportados incluyen .ogg y .mp3.

 player.SetFile( "/Sys/Snd/Poing.ogg" );

Para reproducir un archivo de sonido a partir de su aplicación de sonido activos, el nombre del archivo pasa al método SetFile que debe empezar con "Snd/". Por ejemplo:

 player.SetFile( "Snd/MyAppSound.mp3" );

Cuando el archivo de sonido está listo para reproducir, la función callback OnReady (Listo) será llamada. Usa el método SetOnReady para establecer el nombre de tu función callback OnReady.

Ejemplo

function OnStart()
{
  player = app.CreateMediaPlayer();
  player.SetOnReady( player_OnReady );
  player.SetFile( "/Sys/Snd/Trill.ogg" );
 }

function player_OnReady()
{
  app.ShowPopup( "OnReady" );
  player.Play();
}

El método SeekTo puede ser usado para ajustar la posición del reproductor en el tiempo en segundos. Pasando en 0 estableceremos la posición del reproductor en el principio del archivo de sonido.

 player.SeekTo( 0 );

Cuando el archivo de sonido ha terminado de reproducir, la función callback OnComplete (Completado) será llamado. Usa el método SetOnComplete para establecer el nombre de tu función callback OnComplete.

Ejemplo - OnComplete

function OnStart()
{
  lay = app.CreateLayout( "Linear", "FillXY,VCenter" );

  btn = app.CreateButton( "Play", 0.4, 0.1 );
  btn.SetOnTouch( btn_OnTouch );
  lay.AddChild( btn );

  app.AddLayout( lay );

  player = app.CreateMediaPlayer();
  player.SetOnComplete( player_OnComplete );
  player.SetFile( "/Sys/Snd/Poing.ogg" );
 }

function btn_OnTouch()
{
  player.SeekTo( 0 );
  player.Play();
}

function player_OnComplete()
{
  app.ShowPopup( "OnComplete" );
}

Los siguientes métodos están disponibles en el objeto MediaPlayer:

 SetFile( fileName )
 SetLooping( loop )
 Close()
 Play()
 Pause()
 Stop()
 IsReady()
 IsPlaying()
 IsLooping()
 SeekTo( seconds )
 GetPosition()
 GetDuration()
 SetVolume( left, right )
 SetOnReady( callback )
 SetOnComplete( callback )
 SetOnSeekDone( callback )
