Notificación

El objeto Notification (Notificación) puede ser usado para poner mensajes en la caja de notificaciones.

Crear un objeto de Notificación usando el método CreateNotification (Crear Notificación) del objeto de la app:

 notify = app.CreateNotification( options );

'Options' (opciones) pueden incluir "AutoCancel", "FullScreen" y "Ongoing"

Usa el método SetMessage (Establecer mensaje) para decirle al objeto que mostrar

 notify.SetMessage( ticker, title, text );

El método Notify (notificar) le dice que envíe la notificación

 notify.Notify( id );

Cuando el usuario toca tu notificación, tu aplicación aparecerá aunque no esté corriendo.

Si usas más de una notificación, app.GetNotifyId() puede ser usado para verificar que notificación fue presionada por el usuario.

Ejemplo


function OnStart()
{
  app.ShowPopup("Sending notification");
  notify = app.CreateNotification();
  notify.SetMessage("Ticker","Title","Text");
  notify.Notify("testID");
  setTimeout(cancel,5000);
}

function cancel()
{
  notify.Cancel("testID");
  app.ShowPopup("Notification cancelled");
}

Los siguientes métodos están disponibles para el objeto Notification:

 Notify( id )
 Cancel( id )
 SetMessage( ticker. title, text )
 SetLargeImage( file )
 SetLights( color, onMs, offMs )
