Diálogo Personalizado

Las cajas emergentes de Diálogos Personalizados pueden ser usados para mostrar información temporera o para recolectar datos del usuario. Esto es conveniente para mostrar configuraciones o controles configurables.

Puedes añadir todos los tipos de controles a los objetos CustomDialog. Los diálogos son reajustables y puedes cambiar su apariencia de modo que encajen con el tema de su aplicación.

Crea un objeto de Diálogo Personalizado usando el método CreateDialog de los objetos de la app:

 dlg = app.CreateDialog( title, options );

Las siguientes opciones son válidas para el método CreateDialog:

  • "NoCancel" - El botón 'ATRÁS' no cancela el diálogo
  • "AutoCancel" - Tocar fuera del diálogo cancela la importación
  • "NoTitle" - Elimina el título del diálogo
  • "NoFocus" - Evita el diálogo ganando foco
  • "NoDim" - Evita oscurecimiento fuera de la zona de diálogo.

Un Diálogo Personalizado necesita también un diseño para añadir controles. Usa el método CreateLayout de los objetos de la app para añadir un diseño.

 layDlg = app.CreateLayout( type, options );

Usa el método Show para mostrar el diálogo.

 dlg.Show();

Y el método Dismiss cuando quieres que se cierre.

 dlg.Dismiss();

El siguiente ejemplo muestra como crear un simple CustomDialog en el click de un botón. El diseño para el diálogo es creado y un botón para esconder el diálogo es añadido al diseño.

Ejemplo


function OnStart()
{
  lay = app.CreateLayout( "linear", "VCenter,FillXY" );

  btn = app.CreateButton( "Mostrar Diálogo", 0.6, 0.1 );
  btn.SetOnTouch( btn_OnTouch );
  lay.AddChild( btn );

  app.AddLayout( lay );
}

function btn_OnTouch()
{
  dlg = app.CreateDialog( "Diálogo Personalizado" );

  layDlg = app.CreateLayout( "linear", "VCenter,FillXY" );
  layDlg.SetSize( 0.7, 0.3 );
  dlg.AddLayout( layDlg );

  chk = app.CreateCheckBox( "Casilla de Marcar" );
  chk.SetMargins( 0, 0.02, 0, 0.02 );
  layDlg.AddChild( chk );

  btnDlg = app.CreateButton( "Cerrar Diálogo", 0.6, 0.1 );
  btnDlg.SetOnTouch( btnDlg_OnTouch );
  layDlg.AddChild( btnDlg );

  dlg.Show();
}

function btnDlg_OnTouch()
{
  dlg.Dismiss();
}

Los siguientes métodos están disponibles en el objeto CustomDialog:

 AddLayout( layout )
 RemoveLayout( layout )
 SetTitle( title )
 AdjustColor( hue, sat, bright, const )
 Show()
 Hide()
 Dismiss()
 SetBackColor( clr )
 SetBackground( file, options )
 SetSize( width, height, options )
 SetPosition( left, top, width, height, options )
 SetOnTouch( callback )
 SetOnCancel( callback )
