Descargas

El componente 'Descargas' puede ser usado para descargar un archivo directamente desde Internet hacia el almacenamiento local del teléfono o tableta.

Crea un objeto 'Downloader' usando el método CreateDownloader del objeto de la app:

 dload = app.CreateDownloader( );

El método SetOnComplete es usado para establecer una función para ser llamada cuando la descarga ha terminado o fallado.

 dload.SetOnComplete( callback );

El método 'Download' establece la fuente y el destino del archivo y comienza a descargar.

 dload.Download( srcFileUrl, targetDir );

Una vez que el método de descarga se ha llamado, se muestra una barra de progreso y comienza la descarga.

Ejemplo

function OnStart()
{
  var srcFileUrl="http://sgarman.net/downloadable.txt";
  var targetDir="/sdcard";

  dload = app.CreateDownloader();
  dload.SetOnComplete( dload_OnComplete );
  dload.Download( srcFileUrl, targetDir );
}

function dload_OnComplete()
{
   app.ShowPopup("Descarga Completa");
}
