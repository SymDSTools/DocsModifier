Grabar Discurso

El objeto SpeechRec (Grabar Discurso) puede ser usado para escuchar y reconocer un discurso (diálogo o voz).

Crea un objeto SpeechRec usando el método CreateSpeechRec del objeto de la app:

 speech = app.CreateSpeechRec();

Usa el método Recognize del objeto SpeechRec para decirle que empiece a escuchar:

 speech.Recognize();

Cuando el objeto SpeechRec empiece a escuchar, la función callback OnReady será llamada. Usa el método SetOnReady para establecer el nombre de la función callback OnReady.

Si el objeto SpeechRec reconoce discursos, la función callback OnResult será llamada. Los resultados serán pasados por la función callback OnResult como un array (gama), con el resultado más probable enfrente. Usa el método SetOnResult para establecer el nombre de tu función callback OnResult.

Si el objeto SpeechRec no reconoce nada, la función callback OnError será llamada. Usa el método SetOnError para establecer el nombre de tu función callback OnError.

Ejemplo

function OnStart()
{
  lay = app.CreateLayout( "linear", "VCenter,FillXY" );

  btn = app.CreateButton( "Háblame", 0.3, 0.1 );
  btn.SetOnTouch( btn_OnTouch );
  lay.AddChild( btn );

  app.AddLayout( lay );

  speech = app.CreateSpeechRec();
  speech.SetOnReady( speech_OnReady );
  speech.SetOnResult( speech_OnResult );
  speech.SetOnError( speech_OnError );
 }

function btn_OnTouch()
{
  speech.Recognize();
}

function speech_OnReady()
{
  app.ShowPopup( "Escuchando...", "Short" );
}

function speech_OnResult( results )
{
  app.ShowPopup( results[0] );
}

function speech_OnError()
{
  app.ShowPopup( "Por favor habla más claro!" );
}
