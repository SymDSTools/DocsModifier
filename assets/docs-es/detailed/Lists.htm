Listas

Crea objetos de List usando el método CreateList de los objetos de la app:

 lst = app.CreateList( list, width, height, options );

Usa los métodos SetOnTouch y SetOnLongTouch de los objetos de Listas para establecer el nombre de la función que quieres que sea llamada cuando un artículo de la lista es seleccionado.

El artículo seleccionado title, body, type y index son pasados por la función callback OnTouch como parámetros cada vez que un artículo es seleccionado o se mantiene presionado.

Ejemplo - Simple

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  lst = app.CreateList( "Fred,Bill,Jim", 0.8, 0.4 );
  lst.SetOnTouch( lst_OnTouch );
  lst.SetOnLongTouch( lst_OnLongTouch );
  lay.AddChild( lst );

   app.AddLayout( lay );
}

function lst_OnTouch( title, body, type, index )
{
  app.ShowPopup( "Item = " + title + ", Index = " + index, "Short" );
}
function lst_OnLongTouch( title, body, type, index )
{
  app.ShowPopup( "Long Touch Item = " + title + ", Index = " + index, "Short" );
}

Puedes cambiar la apariencia de una Lista usando las funciones SetBackColor y SetTextColor del objeto Lista.

También puedes establecer una imagen/patrón de fondo o un fondo gradiente para la Lista usando las funciones SetBackground y SetBackGradient.

Ejemplo - Gris en Blanco

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  lst = app.CreateList( "Fred,Bill,Jim", 0.8, 0.4 );
  lst.SetTextColor( "#ff666666" );
  lst.SetBackColor( "#ffffffff" );
  lst.SetOnTouch( lst_OnTouch );
  lay.AddChild( lst );

   app.AddLayout( lay );
}

function lst_OnTouch( title, body, type, index )
{
  app.ShowPopup( "Touched Item = " + title );
}

El objeto Lista también soporta artículos de listas multi-linea y puedes mostrar iconos. Los artículos multi-linea son creados dividiendo cada artículo de la lista usando el carácter ':' (A éste carácter de dos puntos se le conoce en inglés como colon).

Si necesitas usar el carácter colon ':' en el texto de tu artículo después de utilizarlo en el código para crear artículos por separado, necesitas poner este carácter en secuencia: ^c^.

Puedes tener un icono y algunos cuerpo de texto con los siguientes formatos:

title : icon
title : body : icon

Puedes especificar tu propio icono, por ejemplo "Img/MiIcono.png", o uno de los iconos especiales incorporados en esta aplicación usando las siguientes palabras clave:

"folder", "audiofolder", "photofolder", "videofolder", "audio", "photo", "video" and "playlist".

Ejemplo - Título + Icono

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  var data = "Folder:folder,Audio:audio,Photo:photo,Video:video";
  lst = app.CreateList( data, 0.8, 0.4 );
  lst.SetOnTouch( lst_OnTouch );
  lay.AddChild( lst );

   app.AddLayout( lay );
}

function lst_OnTouch( title, body, type, index )
{
  app.ShowPopup( "Touched Item = " + title );
}

También puedes crear listas de los artículos que parezcan botones usando una de las siguientes opciones: "AlumButton", "GreenButton", "OrangeButton".

Ejemplo - Botones Naranjas

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  lst.SetOnTouch( lst_OnTouch );

   app.AddLayout( lay );
}

function lst_OnTouch( title, body, type, index )

O crea listas con fondos Gradientes como éste:-

Ejemplo - Fondo Gradiente

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  var data = "";
  for( var i=1; i<=30; i++)
  {
    if( i>1 ) data += ",";
    data += "Item "+i+":Details for item "+i+":null";
  }
  lst = app.CreateList( data, 1, 1, "WhiteGrad" );
  lst.SetTextColor1( "#ff555558");
  lst.SetTextColor2( "#ff555558" );
  lst.SetTextMargins( 0.04, 0, 0, 0 );
  lst.SetOnTouch( lst_OnTouch );
  lay.AddChild( lst );

   app.AddLayout( lay );
}

function lst_OnTouch( title, body, type, index )
{
  app.ShowPopup( "Touched Item = " + title );
}
