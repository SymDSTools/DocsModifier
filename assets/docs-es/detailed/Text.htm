Texto

Crea objetos de Texto usando el método CreateText de los objetos de la app:

 txt = app.CreateText( text, width, height, options );

Puedes usar la opción "Multiline" para permitir texto en múltiples líneas y las opciones "Left" o "Right" para alinear el texto. El valor predeterminado, es una sola línea de texto centrado.
Las otras opciones que verás utilizados en los ejemplos incluyen AutoScale, Bold, Html y FontAwesome

Usa las funciones SetText y GetText de los objetos de texto para establecer y obtener su texto.

Ejemplo - Línea Simple

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  txt = app.CreateText( "Hello" );
  txt.SetTextSize( 22 );
  lay.AddChild( txt );

   app.AddLayout( lay );
}

Puedes cambiar el aspecto de un texto usando las funciones SetBackColor y SetTextColor en el objeto texto.

Ejemplo - Líneas Múltiples

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  txt = app.CreateText( "Hello\nWorld", 0.8, 0.2, "Multiline" );
  txt.SetTextSize( 12 );
  txt.SetTextColor( "#ff6666ff" );
  txt.SetBackColor( "#ffffffff" );
  lay.AddChild( txt );

   app.AddLayout( lay );
}

El texto se puede acortar usando puntos suspensivos (...) cuando el tamaño del control es muy pequeño para mostrar todo el texto. Usa SetEllipsize para controlar donde se van a colocar los puntos suspensivos. Los modos disponibles son "start", "middle", "end" y "marq".

Ejemplo - Ellipses

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  txt = app.CreateText( "The quick brown fox jumps over the lazy dog", 0.75 );
  txt.SetTextSize( 28 );
  txt.SetEllipsize( "end" );
  lay.AddChild( txt );

   app.AddLayout( lay );
}
