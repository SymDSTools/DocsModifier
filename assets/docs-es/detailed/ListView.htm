Ver Lista

Usted puede crear una pantalla completa para mostrar una lista en la aplicación con el uso del método CreateListView en los objetos de la app:

 lvw = app.CreateListView( list, title, options );

Use el método SetOnTouch del objeto ListView para establecer el nombre de la función que quieres que sea llamado cuando un artículo de la lista es seleccionado. El artículo seleccionado es pasado por la función callback OnTouch como parámetros cada vez que el artículo es seleccionado.

Ejemplo

function OnStart()
{
  lvw = app.CreateListView( "Lun,Mar,Mie,Jue,Vie,Sab,Dom", "Días" );
  lvw.SetOnTouch( lvw_OnTouch );
  lvw.Show();
 }

function lvw_OnTouch( item )
{
  app.ShowPopup( item );
}
