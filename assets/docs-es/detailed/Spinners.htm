Spinner

Crea Spinners usando el método CreateSpinner de los objetos de la app:

 spin = app.CreateSpinner( list, width, height, options );

Puedes permitir que el Spinner ajuste su tamaño automáticamente dejando las dimensiones o puedes especificarlas en ancho y/o alto como fracciones decimales del ancho y el alto de la pantalla.

Usa el método SetOnChange de tu spinner para establecer el nombre de la función que quieres que sea llamada cuando el usuario seleccione un artículo.

Puedes obtener la selección actual del botón Spinner en cualquier momento usando el método GetText, y reemplazar la opción de lista en cualquier momento usando el método SetList.

Usa el método SelectItem para seleccionar un artículo en particular en el código.

Ejemplo

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  spin = app.CreateSpinner( "Bilbo,Frodo,Gandalf", 0.4 );
  spin.SetOnChange( ShowSelection );
  spin.SelectItem( "Frodo" );
  lay.AddChild( spin );

   app.AddLayout( lay );
}

function ShowSelection( item )
{
  app.ShowPopup( "Selected = " + item );
}
