CheckBoxes

Crea CheckBoxes usando el método CreateCheckBox del objeto de la app:

 chk = app.CreateCheckBox( text );

Use el método SetOnTouch del objeto CheckBox para establecer el nombre de la función que será llamada cuando el CheckBox es tocado. Puedes leer el párametro 'isChecked' en la función callback para obtener el estado del CheckBox

Ejemplo

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  chk = app.CreateCheckBox( "Activar la Casilla" );
  chk.SetOnTouch( ShowState );
  lay.AddChild( chk );

   app.AddLayout( lay );
}

function ShowState( isChecked )
{
  app.ShowPopup( "Marcado = " + isChecked, "Short" );
}
