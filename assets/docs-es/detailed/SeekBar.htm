Barra 'Buscar' (SeekBars)

Crea barras de buscar con el método CreateSeekBar de los objetos de la app:

 skb = app.CreateSeekBar( width, height );

El método SetRange del objeto SeekBar para establecer el valor del rango del SeekBar, y el método SetValue para establecer el valor actual.

Usa el método SetOnTouch del objeto del SeekBar para establecer el nombre de una función que quieres que sea llamada cuando el SeekBar es tocado. Puedes leer los parámetros 'value' en tu función callback para obtener la posición del valor del SeekBar

Ejemplo

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  skb = app.CreateSeekBar( 0.8 );
  skb.SetRange( 1.0 );
  skb.SetValue( 0.5 );
  skb.SetOnTouch( skb_OnTouch );
  lay.AddChild( skb );

   app.AddLayout( lay );
}

function skb_OnTouch( value )
{
  app.ShowPopup( "Value = " + value );
}
