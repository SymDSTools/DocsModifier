Botones

Cree botones usando el método CreateButton de los objetos de la app:

 btn = app.CreateButton( text, width, height, options );

Puedes permitir que el botón ajuste el tamaño automáticamente dejando las dimensiones por defecto o especificar una fracción decimal para ancho y alto. Ajuste la anchura y la altura a -1, mientras que el uso de la opción 'FillX' permitirá rellenar el ancho del diseño (Layout).

Use el método SetOnTouch de los objetos de tu botón para establecer el nombre de una función que desees que sea llamada cuando el botón sea tocado.

Varios estilos de botones pueden ser configurados, incluyendo un nombre de estilo de las opciones de parámetros.

Ejemplo - Tamaño por Defecto

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  btn = app.CreateButton( "Tócame" );
  btn.SetOnTouch( SayHello );
  lay.AddChild( btn );

   app.AddLayout( lay );
}

function SayHello()
{
  app.ShowPopup("Traducida por AndroidHackPro!");
}

Ejemplo - Cambiar Estilo

function OnStart()
{
  lay = app.CreateLayout( "Linear", "Vertical,FillXY" );
  lay.SetPadding( 0.1, 0.1, 0.1, 0 );

  b1 = app.CreateButton( "Normal", -1, -1, "FillX" );
  lay.AddChild( b1 );

  b2 = app.CreateButton( "Gray", -1, -1, "FillX,Gray" );
  lay.AddChild( b2 );

  b3 = app.CreateButton( "Alum", -1, -1, "FillX,Alum" );
  lay.AddChild( b3 );

   app.AddLayout( lay );
}

Los siguientes métodos están disponibles en el objeto Botón:

 SetVisibility( visibility )
 GetVisibility()
 SetPadding( left, top, right, bottom )
 SetMargins( left, top, right, bottom )
 SetPosition( left, top, width, height )
 SetSize( width, height )
 GetWidth()
 GetHeight()
 SetOnTouch( callback )
 SetText( text )
 SetHtml( html )
 GetText()
 SetTextColor( colorCode )
 SetTextSize( size )
 GetTextSize( mode )
 SetFontFile( file )
 SetTextShadow( radius, dx, dy, color )
