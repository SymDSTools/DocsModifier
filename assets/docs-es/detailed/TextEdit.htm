Editar Texto

Crear objetos de Editar Texto usado el método CreateTextEdit de los objetos app:

 edt = app.CreateTextEdit( text, width, height, options );

Puedes usar el método SetOnChange del TextEdit para establecer el nombre de la función que quieres que sea llamada cuando el texto sea cambiado.

Usa las funciones SetText y GetText para establecer y obtener el texto en el TextEdit.

Ejemplo - Usando OnChange

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  edt = app.CreateTextEdit( "Editame", 0.8, 0.3 );
  edt.SetOnChange( edt_OnChange );
  lay.AddChild( edt );

   txt = app.CreateText( "", 0.8, 0.3 );
  txt.SetMargins( 0, 0.02, 0, 0 );
  lay.AddChild( txt );

  app.AddLayout( lay );
}

function edt_OnChange()  
{
  txt.SetText( edt.GetText() );
}

Puedes cambiar el estilo del TextEdit usando las funciones SetBackColor y SetTextColor en el objeto TextEdit.

Ejemplo - Azul en Blanco

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  edt = app.CreateTextEdit( "Hello", 0.8, 0.4 );
  edt.SetTextColor( "#ff6666ff" );
  edt.SetBackColor( "#ffffffff" );
  lay.AddChild( edt );

   app.AddLayout( lay );
}
