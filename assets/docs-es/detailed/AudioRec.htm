Grabar Audio

El objeto AudioRecorder (Grabar Audio) puede ser usado para escuchar un sonido y grabarlo en un archivo.

Crea un objeto AudioRecorder usando el método CreateAudioRecorder de los objetos de la app:

 rec = app.CreateAudioRecorder();

Usa el método SetFile para decirle al objeto donde guardará el sonido:

 rec.SetFile("/sdcard/demo.wav");

El método Start (Inicio) le dice que empieze a grabar:

 rec.Start();

Y el método Stop (Detener) es cuando tienes que terminar de grabar:

 rec.Stop();

Ejemplo


function OnStart()
{
  rec = app.CreateAudioRecorder();
  rec.SetFile( "/sdcard/test.wav" );
  rec.Start();
  app.ShowPopup("Por Favor Habla");
  setTimeout(stopit,5000);
}

function stopit()
{
  app.ShowPopup("Grabación Terminada");
  rec.Stop();
}
