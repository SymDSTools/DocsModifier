Sensores

Crear objetos de Sensores usando la función CreateSensor de los objetos de la app:

 sns = app.CreateSensor( type, options );

Los Types (tipos) disponibles son: "Acelerometro", "Campo Magnético", "Orientación", "Luz" y "Proximidad".

Puede usar la función SetOnChange del sensor para establecer el nombre de una función que quieres que sea llamada cuando el cambio (en el sensor) ocurra.

Cambia el rango en el que el sensor comprueba los cambios añadiendo una de las siguientes opciones: "Fastest" (Muy Rápido), "Fast" (Rápido), "Medium" (Mediano) o "Slow" (Lento). "Slow" (Lento) está como el valor predeterminado.

Ejemplo - Acelerometro

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  txt = app.CreateText( "", 0.8, 0.3, "Multiline" );
  lay.AddChild( txt );
  app.AddLayout( lay );

  sns = app.CreateSensor( "Accelerometer" );
  sns.SetOnChange( sns_OnChange );
  sns.Start();

 }

function sns_OnChange( x, y, z, time )
{
  txt.SetText( "x="+x + "\n y="+y + "\n z="+z );
}

Ejemplo - Orientación

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  txt = app.CreateText( "", 0.8, 0.3, "Multiline" );
  lay.AddChild( txt );
  app.AddLayout( lay );

  sns = app.CreateSensor( "Orientation" );
  sns.SetOnChange( sns_OnChange );
  sns.Start();

 }

function sns_OnChange( azimuth, pitch, roll, time )
{
  var msg = " azimuth = " + azimuth.toFixed(1);
  msg += "\n pitch = " + pitch.toFixed(1);
  msg += "\n roll = " + roll.toFixed(1);
  txt.SetText( msg );
}
