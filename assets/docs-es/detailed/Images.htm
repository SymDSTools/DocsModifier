Imagenes

Crea controles de imagenes usando el método CreateImage de los objetos de la app:

 img = app.CreateImage( file, width, height, options );

Use el método SetOnTouch del objeto de la imagen para establecer el nombre de la función que quieres que sea llamado cuando se toque la imagen.

Cuando se tocan los controles de la imagen, se envia un parametro event object a tu función callback que contiene los detalles del toque, por ejemplo, la propiedad action del evento del objeto contiene "Down", "Up" or "Move" cuando el usuario toca y mueve sus dedos por la pantalla y las propiedades X y Y contienen cadenas de las cordenadas de los toques.

Si no estableces un tamaño, el objeto de la imagen coincide con el tamaño original de la imagen. Si estableces una dimensión a un valor positivo, y dejas la otra dimensión como -1, luego la imagen mantendrá su aspect ratio (Relación de Aspecto) original.

Especificando tanto anchura y altura, estirará la imagen para llenar el objeto de la imagen, a menos que utilizes la opción "ScaleCenter" para mantener la imagen en su tamaño original y centrarla en el objeto de la imagen.

Ejemplo - Tamaño Original

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  img = app.CreateImage( "/Sys/Img/Droid1.png" );
  img.SetOnTouch( img_OnTouch );
  lay.AddChild( img );

   app.AddLayout( lay );
}

function img_OnTouch( ev )
{
  if( ev.action=="Down" )
    app.ShowPopup( "Dolió!" );
}

Dibujo de Imágenes

Puedes usar un control de imagen como Canvas (Lienzos) llamando sus métodos de dibujo, como DrawRectangle, DrawCircle, DrawImage etc.

Es posible dibujar sobre una imagen cargada o puede comenzar con una imagen en blanco pasando el valor 'null' a el método CreateImage en lugar de un nombre de archivo. Puedes establecer el color de fondo de tu imagen en blanco usando el método 'SetColor'.

Ejemplo - Dibujar Formas

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  img = app.CreateImage( null, 0.8, 0.8 );
  lay.AddChild( img );

  img.SetColor( "#8888ff" );
  img.SetPaintColor( "#ff0000" );
  img.DrawCircle( 0.5, 0.5, 0.1 );
  img.DrawRectangle( 0.7, 0.7, 0.05 );
  
   app.AddLayout( lay );
}
