Lista de Diálogo

Puedes crear un cuadro de diálogo para mostrar una lista en tu aplicación usando el método CreateListDialog del objeto de la app:

 dlg = app.CreateListDialog(título, listas, opciones);

Usa el método SetOnTouch del objeto de la Lista de Diálogo para establecer el nombre de la función que quieres que sea llamada cuando un artículo de la lista es seleccionado.

El artículo seleccionado es pasado por la función callback OnTouch como parámetros cada vez que es seleccionado.

Ejemplo

function OnStart()
{
  dlg = app.CreateListDialog( "Opciones", "Añadir,Remover,Borrar" );
  dlg.SetOnTouch( dlg_OnTouch );
  dlg.Show();
 }

function dlg_OnTouch( item )
{
  app.ShowPopup( item );
}

Puedes usar la opción "Multi" para crear una Lista de Diálogo con check boxes al lado de cada elemento.

Esto permite múltiples elementos de lista para ser seleccionados. Un parámetro extra isChecked es pasado a la función callback OnTouch cuando un check box es marcado o desmarcado.

Cuando uses la opción "Multi", la Lista de Diálogos pueden ser despedidos/ocultados usando el botón ATRÁS en el teléfono o tableta.

Ejemplo - Multi

function OnStart()
{
  dlg = app.CreateListDialog( "Días", "Lun,Mar,Mie,Jue,Vie,Sab,Dom", "Multi" );
  dlg.SetOnTouch( dlg_OnTouch );
  dlg.Show();
 }

function dlg_OnTouch( item, isChecked )
{
  app.ShowPopup( item + " isChecked = " + isChecked );
}
