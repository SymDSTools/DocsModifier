Botones Palanca

Crea botones palanca usando el método CreateToggle en el objeto de la app:

 btn = app.CreateToggle( text, width, height, options );

Puedes permitir que los botones palanca se ajusten automáticamente dejando las dimensiones o puedes especificar el ancho y el alto como fracciones decimales del ancho y el alto de la pantalla.

Usa el método SetOnTouch del objeto de tu botón para establecer el nombre de la función que quieres que sea llamada cuando el botón sea tocado.

Puedes obtener el estado del botón palanca en cualquier momento usando el método GetChecked del botón. El estado de tu botón puede ser pasado por la función callback OnTouch como un parámetro cada vez que el botón sea tocado.

Ejemplo

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  btn = app.CreateToggle( "Altérname" );
  btn.SetOnTouch( ShowState );
  lay.AddChild( btn );

   app.AddLayout( lay );
}

function ShowState( isChecked )
{
  app.ShowPopup( "Marcado = " + isChecked, "Short" );
}

 GetHeight()

 SetText( text )
